## Installation

You can install `formation` via [pip]:

```bash
pip install formation
```

If napari is not already installed, you can install `formation` with napari and Qt via:

```bash
pip install "formation[all]"
```


To install latest development version:

```bash
pip install git+https://github.com/imhorphen/formation.git
```
