# Installation Napari

## Prerequisite

```
torch with GPU
matplotlib
opencv-python
torch
torchvision
```

## Create your environment and install napari

```
(base) user1> conda create -y -n formation -c conda-forge python=3.11
(base) user1> conda activate formation
(formation) user1> python -m pip install "napari[all]"
```

## Napari

```
(formation) user1> napari
```
